# 🍽️ RecipeFinder – Smart Recipe Search Application

🚀 A web-based recipe search application that allows users to discover delicious recipes based on ingredients, cuisine type, or keywords.

This project demonstrates full-stack development, backend integration, and CI/CD pipeline setup using Jenkins and Docker.

---

## 📌 About The Project

RecipeFinder helps users:

* Search recipes by ingredients
* Explore different cuisines
* View detailed recipe instructions
* Discover cooking inspiration easily

The project combines **frontend development, backend logic, and DevOps practices**.

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/vishalB14/Recipefinder.git
```

### 2️⃣ Navigate to the folder

```
cd Recipefinder
```

### 3️⃣ Run with Python

```
python app.py
```

(Adjust filename if different)

---

## 🐳 Run with Docker

Build the Docker image:

```
docker build -t recipefinder .
```

Run the container:

```
docker run -p 5000:5000 recipefinder
```
